Quanty

Beyond-DFT with localized orbitals Tight-binding

General information

Code homepage: https://www.quanty.org/

Full documentation: https://www.quanty.org/documentation/start

Detailed information

Short description: Quanty is a script language which allows the user to program quantum mechanical problems in second quantization and when possible solve these. It can be used in quantum chemistry as post Hartree-Fock or in one of the LDA++ schemes.
